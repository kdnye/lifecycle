Automated Vendor Routing (Action Matrix)

Instead of manually drafting emails to multiple service providers, the Lifecycle app maps specific intake conditions to targeted Postmark templates.

Stellar Support (support@stellar.tech)

The system routes to Stellar Support strictly for logical access control:

  • Onboarding: Microsoft 365 account creation, license assignment, and distribution list additions.
  • Offboarding: Account disablement, forcing sign-out of all active sessions, license recovery, and converting to shared mailboxes.

Stellar Sales (sales@stellar.tech)

The system routes to Stellar Sales exclusively for physical hardware procurement:

  • Triggered only if the Role Matrix dictates physical hardware (Laptops, Docks, Monitors) is required.
  • Provides shipping destination and necessary model specifications based on department.

BlackPoint (Telecom)

The system routes to BlackPoint for voice systems:

  • Requests for new DIDs (Direct Inward Dialing) numbers.
  • Internal extension assignments.
  • Hardware phone provisioning.
Technical Note: All emails are dispatched via the Postmark REST API. If a ticket is not generated, verify the POSTMARK_SERVER_TOKEN is active in the Google Cloud Run secrets manager.
